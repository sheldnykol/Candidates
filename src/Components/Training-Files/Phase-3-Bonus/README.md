# Bonus Challenges

Complete optional challenges to earn up to **91 extra points**.

## Getting Started

1. **Read**: `BONUS_CHALLENGES.md` for all 10 challenges
2. **Choose**: Pick challenges that interest you
3. **Build**: Implement the features
4. **Test**: Enable `bonus-challenges.test.tsx.disabled` to validate

## Enable Testing

```bash
mv bonus-challenges.test.tsx.disabled bonus-challenges.test.tsx
npm test -- bonus-challenges
```
